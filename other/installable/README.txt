To install this PO System, please ensure that your server has a working installation of Apache, MySQL, and PHP. This can be achieved using an out-of-the-box solution such as XAMPP, MAMP, or WAMP, or you can install them manually from the installer packages.

MySQL (version 5.5): http://dev.mysql.com/downloads/mysql/
Apache (version ): http://httpd.apache.org/download.cgi
PHP (version 5.3.8): http://windows.php.net/downloads/releases/archives/php-5.3.8-nts-Win32-VC9-x86.zip

1. Download the system here.
 
2. Open the .zip file.
 
3. Copy the files from the folder to your server.
 
4. Read the included README.txt file and follow the instructions.
 
5. Run the install script located in the setup folder by navigating to the /setup/install.php file in your web browser from your serverâ€™s root directory.
 
6. Fill in the required information about your server (host name, username, password), our team name, and your school name.
 
7. Fill in the mentor username and password (The mentor can change his/her password later).
 
8. Click Install.
